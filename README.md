# FedEx-Supply-Chain-EDA
# 📦 FedEx Supply Chain Delivery History Analysis (EDA)

## 📖 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **FedEx Supply Chain Delivery History Dataset**. The objective is to analyze shipment records, logistics operations, delivery performance, vendors, manufacturing sites, and transportation methods to extract meaningful business insights and support data-driven decision-making.

---

## 🎯 Business Objective

The main objective of this project is to:

- Analyze historical shipment and delivery data.
- Identify shipment trends and logistics patterns.
- Evaluate delivery performance and delays.
- Understand vendor and manufacturing site contributions.
- Analyze freight costs and shipment values.
- Generate actionable business insights for improving supply chain efficiency.

---

## 📂 Dataset Information

The dataset contains information related to:

- Project Code
- Country
- Shipment Mode
- Vendor
- Product Group
- Manufacturing Site
- Delivery Dates
- Shipment Quantity
- Line Item Value
- Freight Cost
- Delivery Status
- And other supply chain attributes.
